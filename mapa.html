Mapa Conceptual

Algoritmo

Descripción

Es una secuencia ordenada y finita de pasos que permite resolver un problema o realizar una tarea específica.


Características
  • Entrada de datos.
  • Secuencia lógica y ordenada.
  • Produce una salida o resultado.

Ejemplo
Preparar Café Entrada: Agua Café Azúcar Proceso: 1. Hervir agua. 2. Agregar café. 3. Mezclar. 4. Agregar azúcar. Salida: Café listo.

Pseudocódigo

Descripción

Es la representación escrita de un algoritmo utilizando un lenguaje sencillo y estructurado, independiente de cualquier lenguaje de programación.


Características
  • Fácil de leer.
  • Describe la lógica paso a paso.
  • No requiere sintaxis estricta.

Ejemplo
Inicio Escribir "Ingresa un número" Leer numero Si numero > 0 Entonces Escribir "Número positivo" FinSi Fin

Cuadro Sinóptico

1

Análisis

Identificar el problema, comprender los requisitos y definir las entradas y salidas.
2

Diseño

Crear el algoritmo y representarlo mediante pseudocódigo o diagrama de flujo.
3

Implementación

Traducir el algoritmo a un lenguaje de programación.
4

Pruebas

Verificar que el programa funcione correctamente y corregir errores.
5

Mantenimiento

Actualizar el programa, corregir fallas y agregar mejoras.
